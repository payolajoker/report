# 제5장 신도시의 경제 및 생활기반 자족성

## 1. 평가방법론

제5장은 기존의 `자족성` 개념을 보완해 `충족도`를 제안한다. 자족성이 주로 일자리 규모에 집중했다면, 충족도는 신도시 주민의 도시기능 수요가 신도시 안에서 얼마나 해결되는지를 본다.

| 개념 | 쉬운 설명 |
|---|---|
| 경제부문 충족도 | 주민의 일자리 수요가 해당 신도시에서 얼마나 충족되는가 |
| 생활기반 부문 충족도 | 쇼핑·여가 수요가 해당 신도시에서 얼마나 충족되는가 |
| 순충족도 | 단순 총량이 아니라 실제 신도시 내부에서 해결되는 비율 |
| 서울·원도시 의존도 | 해당 기능을 서울 또는 모도시에 기대는 정도 |

## 2. 경제 및 생활기반 충족도 분석

| 항목 | 결과 |
|---|---|
| 경제부문 충족도 | 일자리 증가로 지속적으로 상승 |
| 경제부문 순충족도 | 여전히 낮음. 일자리가 늘어도 신도시 주민의 일자리 수요가 지역 안에서 충분히 해결되지는 않음 |
| 쇼핑·여가 충족도 | 경제부문보다 높음 |
| 1기 신도시 | 목표수준 충족도를 대부분 달성 |
| 2기 신도시 | 목표수준에 미치지 못하며, 특히 경제부문 충족도가 낮음 |

## 2.3 서울 및 원도시 의존도

신도시의 서울 의존도와 원도시 의존도는 시간이 지나면서 모두 감소한다. 다만 1기는 서울 의존도가 원도시 의존도보다 높고, 2기는 원도시 의존도가 서울 의존도보다 높다. 이는 2기 신도시가 서울보다 주변 경기권과 더 강하게 연결되어 있음을 보여준다.

## 3. 경제부문 충족도와 상관성 및 지역 기여효과

| 분석 | 결과 |
|---|---|
| 도시 규모와 경제부문 목표 달성 시간 | 도시 규모가 클수록 경제부문 목표수준 달성에 걸리는 시간이 늘어남 |
| 도시 규모와 생활기반 목표 달성 시간 | 뚜렷한 관련성이 없음 |
| 2기 자족용지 비율과 경제 충족도 | 상관성이 매우 약함 |
| 정책적 의미 | 자족용지를 많이 잡는 것보다 실제 활성화와 업종 유치가 더 중요 |

## 4. 소결

제5장의 가장 중요한 결론은 “자족용지 공급량만으로 자족성을 달성할 수 없다”는 것이다. 실제 입주민의 일자리·쇼핑·여가 수요를 어떤 업종과 시설로 충족시킬지, 그리고 조성 이후 어떻게 활성화할지를 계획해야 한다.

| 결론 | 계획적 시사점 |
|---|---|
| 경제부문 순충족도는 낮음 | 일자리 총량보다 입주민 수요와 맞는 일자리 유치 필요 |
| 쇼핑·여가 충족도는 상대적으로 높음 | 생활서비스는 시간이 지나며 안정화될 수 있음 |
| 2기 신도시는 목표 미달 | 조성 중인 도시의 기능 보완과 장기 추적 필요 |
| 자족용지 비율과 충족도 상관은 약함 | 용지 공급보다 활성화 전략이 핵심 |
| 지역경제 기여 효과 존재 | 신도시는 모도시 사업체·종사자 증가에 실질적으로 기여 |
