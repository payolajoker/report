# 제4장 신도시 주택공급의 주택시장 영향

## 1. 연구방법론

제4장은 신도시가 주택시장에 준 영향을 공급량과 가격 안정화 효과로 나누어 본다. 분석은 주택공급 유형, 주변지역 주택공급 유발, 패널모형, 도시 간 주택가격 인과관계로 구성된다.

| 분석 | 질문 |
|---|---|
| 주택공급 유형 | 1·2기 신도시는 어떤 유형의 주택을 얼마나 공급했는가 |
| 주변지역 공급 유발 | 신도시 개발이 주변 주택공급도 늘렸는가 |
| 패널모형 | 주택공급 증가가 가격 안정화에 기여했는가 |
| 인과관계 분석 | 서울·강남·경기·신도시 간 가격 충격은 어떻게 전파되는가 |

## 2. 주택공급 유형 및 주변지역 주택공급 영향

1기 신도시는 전체 공급에서 임대주택 물량 비중이 높아 주거형평성 효과가 컸다고 해석된다. 2기 신도시는 1기보다 임대주택 물량이 적어, 임대주택 공급 측면의 형평성 기여가 상대적으로 약하다고 본다.

| 구분 | 정리 |
|---|---|
| 1기 신도시 | 단기간 대량 공급, 임대주택 비중 높음, 주거형평성 고려 |
| 2기 신도시 | 공급량은 크지만 장기간·분산 공급, 임대주택 측면의 형평성 효과는 약함 |
| 주변지역 | 신도시 개발은 주변지역의 추가 주택공급도 유발 |

## 3. 주택가격 안정화 효과

### 3.1 패널모형 분석

| 결과 | 해석 |
|---|---|
| 1기 신도시 | 주택공급 증가에 따른 주택가격 안정화 효과가 확인됨 |
| 서울 | 분석기간 동안 비교 도시 중 체계적으로 가격이 낮았던 시기가 나타남 |
| 2기 신도시 | 안정화 효과는 1기보다 약하지만 해당 신도시 가격 안정화에는 기여한 것으로 추정 |
| 한계 | 2기는 개발기간이 길고 공급이 분산되었으며 금융위기 등 외생 요인이 큼 |

### 3.2 도시 간 주택가격 영향/인과관계

그랜저 인과관계와 충격반응함수 분석은 서울·강남 가격 충격이 분당·일산 등에 영향을 주고, 반대로 신도시 가격 충격도 서울·강남에 작게 반응을 만든다는 점을 보여준다. 다만 장기적으로 신도시의 일자리 창출 능력이 약해지면 가격 안정화 기능도 약해질 수 있다고 경고한다.

## 4. 소결

| 결론 | 의미 |
|---|---|
| 1기 신도시는 가격 안정화 효과가 분명함 | 서울 주택가격 급등을 흡수하는 대량 공급 효과가 있었음 |
| 2기 신도시는 효과가 약함 | 장기·분산 공급과 외생 변수 때문에 효과가 선명하지 않음 |
| 필터링 효과 가능성 | 입주 초기에는 주거약자·실수요자에게 상향 이동 기회를 제공했을 가능성 |
| 장기적 한계 | 신도시 고용기능이 약해지면 주택시장 안정 기능도 약해질 수 있음 |
