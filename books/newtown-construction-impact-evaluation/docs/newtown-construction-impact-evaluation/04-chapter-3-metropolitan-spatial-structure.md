# 제3장 광역적 도시공간구조 변화

## 1. 평가 방법론 정립

제3장은 신도시가 서울과 수도권의 공간구조를 어떻게 바꾸었는지 본다. 분석 단위는 인구이동, 통근목적지, 생활권 중심성, 주변지역의 인구·산업 변화, 인구·고용밀도 모형이다.

| 방법 | 데이터·해석 |
|---|---|
| 인구분산 효과 | 서울에서 신도시로 이동한 인구와 수도권 전체 인구변화 비교 |
| 사회연결망 분석 | 통근·통행 데이터를 이용해 생활권과 중심성 변화 파악 |
| 주변지역 영향 분석 | 신도시와 인접지역의 인구, 사업체, 종사자 수 변화 비교 |
| 공간구조 모형 | 서울 CBD와 강남 CBD 거리 기준 인구·고용밀도 경사와 상호작용 분석 |

## 2. 수도권 인구분산 효과 분석

| 구분 | 결과 | 의미 |
|---|---|---|
| 1기 신도시 | 1995년 서울 인구분산 효과 11.9%, 이후 7-8%가 2010년대 초반까지 지속 | 서울 과밀 완화 효과가 비교적 명확함 |
| 2기 신도시 | 인구유입이 주변지역에서 주로 발생 | 서울 직접 분산보다 경기권 내부 재배치 효과가 큼 |
| 수도권 전체 | 서울 인구는 감소했지만 경기도와 수도권 비중은 증가 | 서울 분산과 수도권 집중이 동시에 나타남 |

## 3. 수도권 중심성 변화와 주변지역 영향 분석

생활권은 시간이 지나며 더 잘게 나뉘고 다핵화된다. 분당은 1996년 3개 생활권에서 2016년 16개 생활권으로 크게 분화했고, 판교와 동탄도 각각 2개 생활권에서 4개·6개 생활권으로 분화된 것으로 정리된다.

| 분석 대상 | 변화 |
|---|---|
| 1기 신도시 | 장기 자료가 축적되어 서울 의존 감소와 생활권 분화가 비교적 뚜렷함 |
| 2기 신도시 | 입주와 개발이 진행 중이라 주변지역 전입·통근 구조 변화가 현재진행형 |
| 주변지역 | 신도시 개발의 직접 효과와 주변지역 간접 효과가 경기변동과 함께 나타남 |

## 4. 수도권 공간구조 변화 분석

제3장의 모형 분석은 신도시가 중심지와의 거리, 인접지역, 시기별 경제환경에 따라 서로 다른 효과를 낸다는 점을 보여준다. 즉 신도시는 하나의 정책도구지만, 입지와 시점에 따라 인구분산형, 주변지역 재편형, 고용분산형 효과가 다르게 나타난다.

| 모형 | 질문 | 원문 대조 |
|---|---|---|
| 인구밀도 경사모형 | 서울 중심지와의 거리에 따라 인구밀도 변화가 어떻게 달라졌는가 | [PDF page 291](./assets/pages/pdf-page-291.jpg) |
| 고용밀도 경사모형 | 고용 중심성이 서울 밖으로 얼마나 분산되었는가 | [PDF page 294](./assets/pages/pdf-page-294.jpg) |
| 다중인과 모형 | 신도시와 인접지역 효과가 시간에 따라 어떻게 다른가 | [PDF page 303](./assets/pages/pdf-page-303.jpg) |
| 인구·고용 분산효과 그래프 | 신도시 세대별·지역별 효과 비교 | [PDF page 311](./assets/pages/pdf-page-311.jpg), [PDF page 319](./assets/pages/pdf-page-319.jpg) |

## 5. 소결

| 결론 | 해석 |
|---|---|
| 신도시 효과는 시점별로 다름 | 1990년대 후반과 2000년대 후반 경제위기 등 경기변동과 연동됨 |
| 1기는 서울 인구분산 효과가 뚜렷함 | 서울 과밀 완화 정책으로 일정한 성과가 있음 |
| 2기는 주변지역 영향이 큼 | 서울 분산보다 경기권 내부 이동과 주변지역 재편을 더 많이 만든 것으로 보임 |
| 고용 분산화에도 기여 | 대부분 신도시가 고용 분산 효과를 보이나 산업별 차이가 존재 |
| 보완전략 필요 | 빨대효과와 제로섬 효과를 막기 위해 주변지역 연계 산업·고용 전략 필요 |
