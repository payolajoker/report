# 제5장 3기 신도시 계획기준 수립

## 1. 계획지표 설정

제5장은 보고서의 실무 중심부입니다. 기존 계획기준과 지자체 도시기본계획을 비교해 세대당 인구수, 목표연도, 생활권, 주택·토지·교통·기반시설 기준을 제안합니다.

![제5장 시작과 세대당 인구수 표 PDF page 231](assets/pages/pdf-page-231.jpg)

| 항목 | 기준 |
|---|---|
| 기준연도 | 2020년 |
| 목표연도 | 2030년 |
| 목표연도 세대당 인구 | 2.35명 |
| 적용 대상 | 수도권 지역 공공주택특별법상 330만㎡ 이상 공공주택지구 |

## 2. 계획목표와 핵심가치

| 핵심가치 | 계획 의미 |
|---|---|
| 포용 | 다양한 계층과 세대의 주거권 및 도시권 보장 |
| 자족 | 특화된 자족기능과 도시권 산업생태계 조성 |
| 환경 | 사람과 환경이 공생하는 지속가능하고 회복탄력적인 도시 |
| 안전 | 범죄, 사고, 자연재해로부터 안전한 정주여건 |
| 스마트 | 미래지향적 스마트도시 구축과 운영 |
| 협력 | 시민참여 확대와 협력적 거버넌스 활성화 |

## 3. 부문별 계획기준

### 3.1 분석방법

부문별 기준은 기존 1·2기 신도시, 수도권 사례대상지, 관련 법·제도, 도시개발 트렌드, 3기 신도시 입지 특성을 종합해 제시합니다. 분석대상은 1기 신도시 5개, 2기 신도시 12개, 사례대상지 68개로 총 85개입니다.

### 3.2 공간구조 및 생활권 계획

생활권은 시설 중심 영향권이 아니라 실제 이동 범위로 설정합니다. 보고서는 2016년 가구통행실태조사를 활용해 도보·버스·승용차·광역교통의 평균 이동거리를 기준화합니다.

| 생활권 | 기준 | 설명 |
|---|---:|---|
| 소생활권 | 1km 내외 | 도보 10분 이내 |
| 중생활권 | 3.5km 내외 | 자가용·버스 20분 이내 |
| 대생활권 | 7km 내외 | 광역교통·지하철 30분 이내 |

계획기준은 인접 기성시가지와의 발전축, 도로망, 녹지축을 함께 보고, GTX·S-BRT 같은 대중교통 결절지에는 TOD와 TOC형 개발을 고려하도록 제안합니다.

### 3.3 인구 및 주택계획

3기 신도시는 서울과 가까운 입지, 공공주택특별법, 다양한 지구 규모, 1인 가구 증가를 함께 고려해야 합니다.

| 구분 | 저밀도 | 중밀도(연립·다세대) | 중밀도(아파트) | 고밀도 |
|---|---:|---:|---:|---:|
| 인구밀도 기준 | 150인/ha 이하 | 250인/ha 이하 | 450인/ha 이하 | 450인/ha 초과 |
| 인구배분 비중 | 1-5% 이하 | 5% 이하 | 10-20% | 75-85% |
| 주택건설용지 배분 | 10% 이하 | 5% 이하 | 15-25% | 60-70% |

| 공동주택 규모 | 비중 |
|---|---:|
| 60㎡ 이하 | 40-50% |
| 60-85㎡ 이하 | 45-55% |
| 85㎡ 초과 | 15% 이하 |

| 소유관계 | 비중 |
|---|---:|
| 분양주택 | 65% 이하 |
| 임대주택 | 35% 이상 |

임대주택은 공공주택특별법 시행령의 35% 이상 기준을 반영합니다. 임대주택 규모는 60㎡ 미만 70-80%, 60-85㎡ 30% 이하, 85㎡ 초과 5% 이하로 제시됩니다.

### 3.4 토지이용계획

토지이용의 기본방향은 복합화·입체화, 유연화, 대중교통 중심성 강화입니다. 기존처럼 주거, 상업, 자족, 공공, 공원을 따로 놓는 방식보다 기능이 결합된 복합용도와 입체적 공간을 권장합니다.

| 토지이용 원칙 | 내용 |
|---|---|
| 복합화·입체화 | 주거, 상업·업무, 자족시설, 문화여가, 공공시설, 공원·녹지 결합 |
| 유연화 | 획일적 비율보다 지구 특성과 변화 가능성 반영 |
| 대중교통 중심성 | S-BRT, GTX, 지하철 역사 주변 TOD 도입 |
| 자족용지 | 신도시 전체 면적 10% 이상, 규모·입지에 따라 가용면적 대비 비율 상향 |

| 규모 유형 | 면적 기준 | 자족용지 비율 |
|---|---:|---:|
| 대형 | 660만㎡ 이상 | 10%(20%) 이상 |
| 중형 | 330만㎡ 이상 | 10%(15%) 이상 |
| 소형 | 330만㎡ 미만 | 10%(10%) 이상 |

보고서는 자족용지를 외곽 제조업 공간으로만 두지 말고, 중심부에는 고부가가치 서비스업·첨단산업·공유오피스·복합용도를, 외곽부에는 중·대형 필지와 생산·연구·물류·스마트팜 등을 배치하는 방식을 제시합니다.

### 3.5 교통시설계획

교통 기준의 핵심은 자동차 중심 도로망에서 사람 중심, 대중교통 중심으로 바꾸는 것입니다.

| 항목 | 기준 |
|---|---:|
| 전체 도로율 | 도시 면적 대비 15-18% |
| 보행자전용도로 | 1% 이상 |
| BRT 전용차로 | 양방향 6.5m + 중앙분리대 1.5m, 총 8m 이상 |
| 대중교통 분담률 목표 | 40% 이상 |
| 주거·상업 이면도로 속도 | 안전속도 5030과 교통정온화 적용 |

| 도로위계 | 도로율 | 도로 내 구성비 |
|---|---:|---:|
| 주간선도로 | 2-3% | 12-20% |
| 보조간선도로 | 3-5% | 17-33% |
| 집산도로 | 4-5% | 22-33% |
| 국지도로 | 4-5% | 22-33% |
| 보행자전용도로 | 1% 이상 | 5-8% |
| 기타 | 1% 내외 | 5-6% |

대중교통은 GTX, S-BRT, BTX를 중심으로 `trunk-and-feeder` 체계를 만들고, 보행·자전거·PM을 First-Mile/Last-Mile 수단으로 연결하도록 제안합니다.

### 3.6 기반시설계획

기반시설계획은 공원녹지, 학교, 공공청사, 복합커뮤니티시설, 사회복지시설, 문화체육시설, 보건위생시설, 스마트 도시기반시설, 공급처리시설을 포괄합니다.

| 시설 | 주요 기준 |
|---|---|
| 공원녹지 | 지구 내 환경가치 높은 토지 원형보전, 최소 20% 이상 공원녹지 확보 |
| 공원 접근성 | 공원으로부터 도보 10분 또는 반경 500m 내 접근 가능하도록 배치 |
| 초등학교 | 3,500-4,500세대당 1개소 |
| 주민센터 | 30,000-50,000명당 1개소 |
| 소방서 | 100,000명당 1개소 |
| 경찰서 | 100,000명당 1개소 |
| 우체국 | 50,000-100,000명당 1개소 |
| 종합병원 | 필요 시 1-2개소 |

기반시설의 중요한 방향은 복합화입니다. 학교, 공공청사, 생활체육, 보육, 복지, 문화시설을 따로 흩어 놓기보다 생활권 중심에서 복합커뮤니티 기능을 만들도록 합니다.

### 3.7 환경 및 에너지 계획

환경·에너지 기준은 보전, 열섬 저감, 정온환경, 자원순환, 친환경에너지로 나뉩니다.

| 영역 | 기준 방향 |
|---|---|
| 환경·생태 보전 | 그린벨트 환경평가 1·2등급지는 원형보전 원칙 |
| 도시열섬 저감 | 바람길, 입체녹화, 친수공간, 수변 녹도 |
| 정온환경 | 주거·교육시설은 소음·악취 유발원과 이격, 최소 65dB 이하 정온환경 |
| 미세먼지 대응 | 공원녹지, 입체녹화, 대중교통 확대, 차도-녹지-보도 설계 |
| 자원순환 | 빗물·중수 활용, LID, 투수성 포장, 폐기물 재활용 |

### 3.8 방재 및 방범계획

방재계획은 폭우, 폭염, 지진, 폭설, 황사, 미세먼지 등에 대응합니다. 토지이용 단계에서 취약지역을 파악하고, 기반시설·건축물·운영관리까지 연결해야 합니다.

| 구분 | 주요 기준 |
|---|---|
| 토지이용 | 하천변 완충지대, 급경사지·산사태 위험지 피복·식재 |
| 기반시설 | LID, 우수관·하수관 역류 방지, 저류형 도시공원, 대피로와 대피시설 |
| 건축물 | 침수위 고려 출입구, 필로티, 지하저류조, 바람길 고려 배치 |
| 운영관리 | 도시통합운영센터 연계 재해관리시스템, 조기경보, 취약계층 보호 |
| 방범 | CPTED, 자연감시, CCTV, 학교 반경 500m 유해 용도 제한 |

### 3.9 기타 계획

기타 계획은 도시브랜드, 경관계획, 공공디자인계획을 다룹니다.

| 항목 | 방향 |
|---|---|
| 도시브랜드 | 포용성, 자족성, 환경성, 안전성, 스마트, 협력 등 공통 가치와 지구별 개별 가치 반영 |
| 경관계획 | 3D 도시계획, 디지털트윈, VR·AR 기반 경관 시뮬레이션 활용 |
| 공공디자인 | 생활안전, 유니버설디자인, 보행자 우선, 협력적 디자인 프로세스 |

## 제5장의 의미

이 장은 3기 신도시를 실제로 계획할 때 필요한 `숫자`, `공간 원칙`, `시설 배치 기준`, `운영 기준`을 한곳에 모아 둔 핵심 장입니다. LLM이 이 책에서 가장 많이 참조해야 할 장도 제5장입니다.
