# 연구요약

## 연구 배경과 목적

이 연구는 서울과 30분 이내의 근거리에 5개의 대규모 공공택지지구를 조성해 총 17.3만호를 공급하려는 3기 신도시 정책을 전제로 합니다. 문제의식은 분명합니다. 1·2기 신도시에서 반복된 자족기능 부족, 생활인프라 부족, 교통 지연, 주변 난개발을 다시 만들지 않도록, 새로운 개발목표와 계획기준을 먼저 세우자는 것입니다.

## 기존 신도시에서 얻은 교훈

| 구분 | 요약 |
|---|---|
| 추진체계 | 1기는 범정부 추진기구 중심, 2기 이후는 부처 중심, 3기는 공공주택특별법 기반으로 전환 |
| 사업방식 | 중앙집중형 개발에서 지자체·지방공사 참여가 커지는 협력형 개발로 이동 |
| 반복 문제 | 자족성 부족, 주변지역 난개발, 입주시기와 기반시설 공급시기 불일치, 총량·물리 지표 중심 계획 |
| 반영 방향 | 광역 지역개발 틀, 주변지역 연계, 자족성 강화, 인구구조 변화 대응, 스마트사회 구현 |

## 3기 신도시 지정현황

| 지구 | 면적 | 계획호수 | 비고 |
|---|---:|---:|---|
| 남양주 왕숙1·2 | 1,134만㎡ | 66천호 | 대규모 신도시 |
| 하남 교산 | 649만㎡ | 32천호 | 대규모 신도시 |
| 인천 계양 | 335만㎡ | 17천호 | 대규모 신도시 |
| 고양 창릉 | 813만㎡ | 38천호 | 대규모 신도시 |
| 부천 대장 | 343만㎡ | 20천호 | 대규모 신도시 |
| 과천 | 155만㎡ | 7천호 | 중규모 지구로 별도 언급 |

![3기 신도시 현황 표와 위치도 PDF page 033](assets/pages/pdf-page-033.jpg)

보고서는 3기 신도시의 입지 특성을 서울 경계와 매우 가까운 신규택지로 봅니다. 요약부에서는 서울 경계로부터의 거리를 1기 신도시 5km, 2기 신도시 10km, 3기 신규택지 2km로 비교합니다.

## 개발여건 분석 방법

| 분석 | 내용 |
|---|---|
| 포털·언론 관심도 | 구글 트렌드, 네이버 데이터랩, 빅카인즈를 활용해 2018.11 이후 검색·뉴스 추이 분석 |
| 오피니언 분석 | 전국적 보급망을 가진 100대 신문사의 전문가 오피니언 기사 분석 |
| 전문가 설문 | 도시개발 관련 전문가 110명 대상 |
| 주민 설문 | 신도시 및 주변지역 주민 806명 대상 |

## 비전과 목표

보고서가 제시한 3기 신도시의 비전은 `공간혁신과 공생발전의 미래도시`입니다. 이를 위해 6개 계획목표를 둡니다.

| 목표 | 의미 |
|---|---|
| 산업과 일자리 | 자족기능과 고용 기반 강화 |
| 편리한 교통 | 서울 및 주변도시 접근성 개선 |
| 교육과 환경 | 양질의 교육·보육, 친환경 정주여건 |
| 협력과 거버넌스 | 지자체·전문가·주민 참여 |
| 공간과 기능혁신 | 콤팩트·네트워크·스마트 도시 구현 |
| 포용과 공생 | 계층·세대·지역 간 상생 |

## 계획기준의 적용 대상과 구성

계획기준 적용 대상은 수도권 내 330만㎡ 이상의 공공주택지구 개발사업입니다. 구성은 총칙, 계획목표와 핵심가치, 부문별 계획기준, 부록으로 나뉩니다.

| 묶음 | 세부 내용 |
|---|---|
| 총칙 | 목적, 적용범위, 계획지표 |
| 핵심가치 | 포용, 자족, 환경, 안전, 스마트, 협력 |
| 부문별 기준 | 공간구조·생활권, 인구·주택, 토지이용, 교통시설, 기반시설, 환경·에너지, 방재·방범, 기타 계획 |
| 부록 | 용어정의 |

## 요약의 핵심 결론

3기 신도시는 주택공급량만으로 평가하면 안 됩니다. 이 보고서의 핵심은 대규모 공급을 하되, 서울 의존을 줄이고, 주변도시와 연결하며, 신도시 내부의 자족·환경·안전·스마트 운영 능력을 처음부터 계획기준으로 박아 넣어야 한다는 것입니다.
