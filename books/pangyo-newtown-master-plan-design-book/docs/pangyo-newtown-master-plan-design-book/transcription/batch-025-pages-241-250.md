# PDF pp.241-250 - 턴키 프로젝트: A26,27-1 Block

Coverage: this batch is within `03 판교신도시 건축계획안`, `턴키 프로젝트`, and documents the A26,27-1 Block proposal by Daelim Industrial and Woori Construction, presented as `판교휴먼시아 e-편한세상 (2공구)`.

## PDF p.241

![PDF page 241](../assets/pages/pdf-page-241.jpg)

- Project heading: `PANGYO HOUSING Competition`, `A26,27-1 BLOCK`.
- Developer/contractor line: `대림산업 주식회사+우림건설(주)`.
- Design office: `(주)희림 종합건축사사무소`.
- Listed names: `정영균 / 박규용 / 이문림 / 이동원 / 황규익 / 서원호 / 장성근 / 송승원 / 권정일 / 박성민 / 유영무 / 류종현 / 김승주 / 김주희`.
- Main concept heading: `자연이 함께하고 삶이 풍요로운 곳...Enjoy of LOHAS`.
- Text describes three planning directions:
  - Nature-friendly space linked to the Dong-Pangyo water-circulation system and nearby neighborhood park.
  - Human-friendly community space through community facilities, plazas, safe pedestrian space, and barrier-free planning.
  - Landscape-friendly layout and residential-building planning to secure view corridors and near/mid/far landscape experiences.
- A26 block narrative notes view corridors linking existing northern complexes, middle/high schools, and detached-housing areas; tower and angled building forms are used for openness.
- A27 block narrative notes that tower-type building placement is used to improve household views and create a more open outdoor spatial structure; community facilities are organized around family/community plazas.
- Bottom diagrams/captions:
  - `Blue Network` and `Green Network`; bullets reference Dong-Pangyo water-circulation and green-network/view-corridor formation toward the neighborhood park.
  - Pedestrian/community diagram; bullets reference community facilities and plazas along the complex pedestrian route, plus safe barrier-free pedestrian space.
  - Landscape/view diagram; bullets reference maximizing views and securing view corridors.
